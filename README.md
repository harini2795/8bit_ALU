# 8-bit ALU in Verilog/SystemVerilog

This project implements an 8-bit ALU supporting arithmetic, logical, shift, multiplication, and comparison operations. 

## Features
- ADD, SUB, AND, OR, XOR, SHL, SHR, MUL, SLT, EQUAL
- Flags: Zero (Z), Negative (N), Carry (C), Overflow (V)
- Self-checking SystemVerilog testbench
- Modular design, easy to extend

## How to Run
- Open the files in Xilinx Vivado/ModelSim
- Simulate `tb_alu_8bit.sv` to see all operations tested automatically
